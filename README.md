# tricura_take-home-assignment_ml-project_260309
Data science take-home project exploring incident risk prediction and prevention using healthcare operational data.

## Data

The dataset used in this project was provided as part of the Tricura take-home assignment
and is therefore not included in this repository.

To run the notebook:

1. Download the dataset provided by Tricura.
2. Place the `.parquet` files in the following directory:

data/raw/

3. Update the `DATA_PATH` variable in the notebook if necessary.
